· ① 인증하기 클릭 후 활성화

비밀번호
· 빈 값이 디폴트, 마스킹(*) 처리
· 눈 아이콘 클릭 → 마스킹 토글
· 포커스(입력 시작) 시 우측에 실시간 체크리스트 노출
· 조건 충족 시 실시간으로 ☐ → ☑ 변경
· 허용되지 않는 문자 입력 시 즉시 삭제 + 툴팁 안내 (2~3초 자동 사라짐)
· 모든 항목 충족 시 체크리스트 자동 숨김
· 미충족 상태에서 다음 필드 이동 시: "비밀번호 조건을 확인해주세요"

약관·동의 운영 기준
① 개인정보 동의 — 목적별 분리 운영 (개인정보보호법 제17~21·27·28조)
필수계정 운영 — 서비스 제공에 필요한 개인정보 수집·이용
선택마케팅 — 광고·프로모션 목적 활용
해당 시 필수제3자 제공 — 제공처·이용목적·항목을 동의서에 개별 명시
해당 시 필수국외 이전 — 이전 국가·제공처·이용목적·항목을 동의서에 개별 명시
② 약관·서면 5개 그룹 — 그룹별 분리 관리
(1) 플랫폼 서비스 이용약관
(2) 금융상품 거래 관련 약관 — 금상법 제37조의3, 계약체결전 교부서면
해당 시(3) 외국증권 거래계좌 약관 / 취급규정
(4) 개인정보 수집·이용·제3자 제공·국외 이전 동의 — ① 항목 참조
(5) 반사회적 세력 해당 없음 확약서 + PEP 비해당 확인동의
· PEP(Politically Exposed Person): 초기 단계 이용 대상 제외 방침을 약관에 명시, 가입 시 비해당 확인동의 수령
· 하이리스크 거래 범위: 북한·이란 등 제재 대상국 국적자와의 거래 포함
③ 개발 체크포인트
필수중요항목 개별 체크박스 — 금융분야 개인정보보호 GL 제3조. 중요 동의항목은 일괄 동의 방식 불가, 항목별 개별 체크박스 구현 필수
