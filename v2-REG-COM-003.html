법인소재지
· 디폴트: 일본(🇯🇵) 선택 상태
· 드롭다운으로 국가 변경 가능
· 일본 선택 시 → 현재 화면(REG-COM-003) 유지
· 일본 외 국가 선택 시 → 해외 법인정보입력 화면(REG-COM-005-overseas)으로 전환
· 국가 드롭다운: 상위 7개국(일본, 싱가포르, 중국, 한국, 홍콩, 대만, 호주) 우선 노출 → 구분선 → 나머지 가나다순, 검색 기능 포함
2
법인번호 + 조회
· 빈 값이 디폴트
· 숫자만 입력 가능, 13자리 고정
· 13자리 입력 완료 시 조회 버튼 활성화
· 조회 클릭 → 국세청(法人番号公表サイト) API 호출
· 조회 성공 → ③ 조회결과 카드 노출
· 조회 실패 시 → "법인번호 조회에 실패했습니다" + 직접입력 모드 전환 링크 노출
· 직접입력 모드: 법인명(한자), 법인명(카타카나), 본점주소를 수동 입력하는 필드 노출
3
조회결과 카드
· ② 조회 성공 시에만 노출
· 표시 항목: 법인명(한자), 법인명(카타카나), 본점주소
· 읽기전용 — 사용자 수정 불가
· 법인번호 재입력 시 카드 초기화
· 직접입력 모드에서는 카드 대신 편집 가능한 입력 필드 노출
4
상장여부
· 적격기관 리스트에 없는 법인에게만 표시 — 리스트 등재 법인은 내부 DB에서 자동 확인하므로 미표시
· 미선택이 디폴트
· 상장 / 비상장 중 1개만 선택 가능
· 상장 선택 → ⑤ 자본금 영역 숨김, 다음 버튼 활성화 (UBO 입력 패스)
· 비상장 선택 → ⑤ 자본금 영역 노출
5
자본금
· 적격기관 리스트에 없는 법인 + 비상장 선택 시에만 노출
· 미선택이 디폴트
· 5억엔 미만 / 5억엔 이상 중 1개만 선택
· 5억엔 이상 → 특정투자자 자격 인정, UBO 입력(REG-COM-005)으로 이동
· 5억엔 미만 → 자격 없음, 매도 불가 안내(REG-COM-012)로 이동
6
다음 버튼
· 비활성이 디폴트, 비활성 시 클릭 불가
· 활성 조건: ② 법인번호 조회 완료(또는 직접입력 완료) + 적격기관 X인 경우 ④ 상장여부 선택 + (비상장 시) ⑤ 자본금 선택
· 클릭 시 적격기관 판정 결과에 따라 분기 → 상세내용 참조
케이스별 분기 (요약)
· 적격기관 O + 상장 → UBO 패스, 담당자정보입력(REG-COM-006)
· 적격기관 O + 비상장 → UBO 입력(REG-COM-005)
· 적격기관 X + 상장 → UBO 패스, 담당자정보입력(REG-COM-006)
· 적격기관 X + 비상장 + 5억엔 이상 → UBO 입력(REG-COM-005)
· 적격기관 X + 비상장 + 5억엔 미만 → 매도 불가 안내(REG-COM-012)
특정투자자 판정 로직 (일본 법인)
법인번호 입력 → 적격기관 리스트 확인 (내부 DB 자동)
  ├─ 적격기관 O → 상장 여부 내부 DB 자동 확인
  │     ├─ 상장 → UBO 패스 → 담당자정보입력(REG-COM-006)
  │     └─ 비상장 → UBO 입력(REG-COM-005) → 담당자정보입력
  └─ 적격기관 X → 화면에서 상장 여부 직접 질문
       ├─ 상장 → UBO 패스 → 담당자정보입력(REG-COM-006)
       └─ 비상장 → 자본금 질문
              ├─ 5억엔 이상 → UBO 입력(REG-COM-005) → 담당자정보입력
              └─ 5억엔 미만 → 매도 불가 안내(REG-COM-012)
※ 법인 조합(투자조합 O)은 법인 판정 결과를 자동 승계 — 별도 자격 확인 없음
비고
· 이 화면은 법인소재지가 일본인 경우의 법인정보 입력 화면입니다.
· 법인소재지를 일본 외로 변경하면 해외 법인정보입력으로 전환됩니다.
· 조회 실패 시 직접입력 모드로 전환하여 법인명/주소를 수동으로 입력할 수 있습니다.
· 다음 화면: UBO 입력(REG-COM-005) / 담당자정보입력(REG-COM-006) / 매도 불가 안내(REG-COM-012)

[공통 규칙] 순차 입력

· 아직 차례가 아닌 칸은 회색(#ccc) 줄 유지
