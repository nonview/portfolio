PIN 4자리 입력
· 빈 값이 디폴트, 4개의 개별 입력칸
· 첫 번째 칸에 자동 포커스 (검은색 테두리)
· 숫자만 입력 가능 (0~9)
· 1자리 입력 시 자동으로 다음 칸으로 포커스 이동
· 백스페이스 시 이전 칸으로 포커스 이동 + 값 삭제
· 4자리 모두 입력 시 확인 버튼 활성화
· 숫자 외 문자 입력 시 무시 (입력 불가)

확인 버튼
· 비활성이 디폴트 (회색 테두리)
· 4자리 모두 입력 시 활성화 (파란색)
· 클릭 시 PIN 검증 API 호출
· 성공 시 → 거래소 홈으로 이동 (PIN 배너 제거, 거래 가능 상태)
· 에러 메시지 3종 → 상세내용 참조
에러 메시지
· 불일치: "PIN 번호가 올바르지 않습니다. 다시 확인해 주세요" + 입력값 초기화
· 5회 실패: "PIN 인증 시도 횟수를 초과했습니다. 고객센터에 문의해 주세요" + 입력 비활성화
· PIN 만료: "PIN이 만료되었습니다. 새로운 PIN을 요청해 주세요" + 재발송 버튼 노출
