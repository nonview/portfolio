1
수입 항목 (복수 선택)
· 미선택이 디폴트
· 복수 선택 가능 (토글 방식: 탭하면 선택, 다시 탭하면 해제)
· 선택 시 파란 테두리 + 파란 텍스트 + 연파란 배경
· 최소 1개 이상 선택 필수
· 선택 항목: 근로소득, 사업소득, 주식부동산양도소득, 부동산임대소득, 이자배당소득, 연금소득, 기타소득
· 요건 판정에 직접 영향 없음 (참고 정보)
2
작년 수입 총합
· 미선택이 디폴트
· 3개 중 1개만 선택 (라디오 방식)
· 선택지: 1억엔 이상 / 1천만엔~1억엔 미만 / 1천만엔 미만
· 요건 판정 핵심: 1천만엔 이상이어야 투자 가능 요건 1차 충족
· 1천만엔 미만 선택 시 경력 요건(④)으로 보완 가능
3
투자경험
· 미선택이 디폴트
· 2개 중 1개만 선택 (라디오 방식)
· 선택지: 1년 안됐어요 / 1년 넘었어요
· 요건 판정 핵심: 1년 이상이어야 투자 가능 요건 충족
· 1년 미만 선택 시 → 요건 미충족 (REG-IND-003)으로 이동
4
관련 경력 (단일 선택)
· 미선택이 디폴트
· 라디오 방식 — 1개만 선택 가능, 선택 즉시 다음 버튼 활성화
· 선택 항목: 은행/보험/증권, 경영전략, 자금조달, 변호사/회계사 등, 상장회사 임원, 경제/경영 교수, 경영컨설턴트, 해당없음
· 요건 판정 보완: 연수입 1천만엔 미만이더라도 관련 경력 선택(해당없음 제외) 시 요건 충족 가능
5
다음 버튼
· 비활성이 디폴트 (회색 테두리 + 회색 글씨)
· ①~④ 모든 항목 입력 완료 시 활성화 (파란 배경 + 흰 글씨)
· 클릭 시 요건 판정 로직 실행 (상세 내용 참조)
이동 분기
통과 → 고객정보입력(REG-IND-004)으로 이동
미충족 → 매도만 가능 안내(REG-IND-003)으로 이동
요건 판정 로직
통과 조건 (OR):
· Case A: 연수입 1억엔 이상 + 투자경험 1년 이상
· Case B: 연수입 1천만엔~1억엔 미만 + 투자경험 1년 이상
· Case C: 연수입 1천만엔 미만 + 투자경험 1년 이상 + 관련 경력 1개 이상

미충족 조건:
· Fail A: 투자경험 1년 미만 (연수입/경력 무관)
· Fail B: 연수입 1천만엔 미만 + 경력 없음 (투자경험 1년 이상이어도)
※ 미충족 후 분기: REG-IND-003에서 매도 의향 재확인
  · Yes → 매도 전용 계정으로 고객정보입력(REG-IND-004) 진행
  · No → 가입 종료
비고
· 이 화면은 가입목적선택(REG-IND-001)에서 "투자" 선택 후 팝업 확인 시 진입합니다.
· 설문 항목은 단일 화면에 모두 노출 (스크롤 가능)
· 요건 판정은 서버 사이드에서 수행, 클라이언트는 결과만 수신
· 뒤로가기 → 가입목적선택(REG-IND-001)으로 복귀
