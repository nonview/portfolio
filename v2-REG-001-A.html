유저 유형 선택
· 미선택이 디폴트, 1개만 선택, 선택 후 변경 가능
· 개인 선택 시 → 개인 질문 영역 노출 (페이지 B)
· 법인 선택 시 → 법인 질문 영역 노출 (페이지 C)
선택에 따른 분기
· 개인 선택 시 → 개인 질문 영역 노출 (REG-001-B 참조)
· 법인 선택 시 → 법인 질문 영역 노출 (REG-001-C 참조)
비고
· 가입 플로우 최초 진입 화면 — 아무것도 선택되지 않은 초기 상태
· 개인/법인 선택 전까지 추가 질문 및 하단 버튼 영역 미노출
· 선택 후 변경 가능 (개인 ↔ 법인 재선택 허용)
