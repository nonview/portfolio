· 고객정보입력: 완료 상태 (녹색 체크)

· 스텝 사이 연결선으로 진행 상태 표시
2
QR코드 영역
· 안내 문구 + QR코드 이미지가 디폴트
· QR코드는 eKYC 본인확인 서비스(외부) 연동 URL을 인코딩
· 사용자가 스마트폰 카메라로 스캔 → 외부 eKYC 앱/웹으로 이동
· 에러 메시지 2종 → 상세내용 참조
3
안내 문구
· "스마트폰에서 본인확인절차를 마치신 후 아래 버튼을 눌러 주세요" 고정 텍스트
· 사용자에게 스마트폰 eKYC 완료 후 PC에서 완료 버튼 클릭을 유도
4
완료 버튼
· 녹색 활성 상태가 디폴트
· 클릭 시 서버에 eKYC 완료 여부 조회
· eKYC 완료 확인됨 → 본인확인 완료 화면(SCR-KYC-002)으로 이동
· 에러 메시지 2종 → 상세내용 참조
에러 메시지
· 유효시간 만료: "QR코드가 만료되었습니다. 새로고침해 주세요" + 새로고침 버튼 노출
· 생성 실패: "본인확인 서비스에 연결할 수 없습니다" + 재시도 버튼
에러 메시지
· eKYC 미완료: "본인확인이 아직 완료되지 않았습니다. 스마트폰에서 절차를 완료해 주세요"
· eKYC 실패: "본인확인에 실패했습니다. 다시 시도해 주세요" + QR코드 재생성
선택에 따른 분기
· 다음 화면: 본인확인 완료 (SCR-KYC-002)
비고
· 이 화면은 고객정보입력 완료 후 eKYC 본인확인을 진행하는 화면입니다.
· QR코드 스캔 방식으로 스마트폰의 eKYC 인증 절차를 시작합니다.
· eKYC 서비스는 외부 연동이며, 신분증 촬영 + 얼굴 인식 등의 절차가 스마트폰에서 진행됩니다.
