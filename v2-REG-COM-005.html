1
실질적지배자 유형
· 미선택이 디폴트
· 개인 / 법인 / 개인 및 법인 중 1개만 선택 가능
· 개인 선택 시 → ② 개인 지배자 입력 섹션만 노출
· 법인 선택 시 → ③ 법인 지배자 입력 섹션만 노출
· 개인 및 법인 선택 시 → ② + ③ 모두 노출
· 유형 변경 시 기존 입력값 초기화 후 해당 섹션 재노출
2
개인 지배자 입력
· ①에서 개인 또는 개인 및 법인 선택 시 노출
· 입력 필드: 국적(드롭다운), 성명한자, 성명카타카나, 생년월일, 거주지주소
· 국적 디폴트: 일본(🇯🇵)
· 생년월일: YYYY/MM/DD 형식, 날짜 피커 제공
· 거주지주소: 우편번호 입력 → 자동 주소 검색 + 나머지 수동 입력
· "+ 개인 지배자 추가" 클릭 → 동일 입력 폼 1세트 추가 (최대 4명까지)
· 추가된 지배자 우측에 삭제(X) 버튼 노출
· 최소 1명 필수
3
법인 지배자 입력
· ①에서 법인 또는 개인 및 법인 선택 시 노출
· 입력 필드: 법인소재지(드롭다운), 법인명한자 + 조회 버튼
· 조회 클릭 → 법인번호 공표사이트 API 조회
· 조회 성공 시 법인명(카타카나), 본점주소 자동 표시
· 조회 실패 시 직접입력 모드 전환
· "+ 법인 지배자 추가" 클릭 → 동일 입력 폼 1세트 추가
· 추가된 법인 우측에 삭제(X) 버튼 노출
· 최소 1법인 필수
