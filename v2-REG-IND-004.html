· 한자 입력 시 자동 변환 불가 — 직접 입력 필요

· 만 18세 미만 선택 불가: "만 18세 이상만 가입 가능합니다"
· 필수 선택

직업
· "선택해 주세요"가 디폴트

· "기타" 선택 시 직접 입력 필드 노출
· 필수 선택
8
휴대폰번호
· 빈 값이 디폴트, 국가코드 +81 고정 표시
· 숫자만 입력 가능, 하이픈 자동 삽입
· 일본 휴대폰번호 형식 체크 (090/080/070으로 시작, 11자리)
· 형식 미충족 시: "유효한 휴대폰번호를 입력해 주세요"
· 필수 입력

· 빈 값이 디폴트
· 우편번호 입력 + [검색] 버튼 클릭 → 도도부현/시구정촌 자동 입력
· 우편번호: 7자리 숫자 (XXX-XXXX 형식, 하이픈 자동 삽입)
· 검색 결과 없을 시: "해당 우편번호를 찾을 수 없습니다"
· 나머지 주소: 직접 입력 (번지, 건물명 등)
· 필수 입력

확인 버튼
· 비활성이 디폴트 (회색 테두리 + 회색 글씨)
· ①~⑨ 모든 필수 항목 입력 완료 시 활성화 (파란 배경 + 흰 글씨)
· 클릭 시 입력 데이터 유효성 검증 후 본인확인(REG-IND-005)으로 이동
· 유효성 미충족 시 해당 필드에 에러 메시지 표시, 화면 이동 없음

· "기타" 선택 시 직접 입력 필드 노출
