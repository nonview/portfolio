1
법인 등기부등본
· 미첨부가 디폴트
· 허용 파일 형식: PDF, JPG, PNG (최대 10MB)
· 업로드 완료 시 파일명 + 삭제(X) 버튼 노출
· 해당 국가에서 발행된 법인 등기 관련 공적 서류
2
법인 인감증명서
· 미첨부가 디폴트
· 허용 파일 형식: PDF, JPG, PNG (최대 10MB)
· 업로드 완료 시 파일명 + 삭제(X) 버튼 노출
· 인감증명서가 없는 국가의 경우 서명 공증 서류로 대체 가능
3
위임장
· 미첨부가 디폴트
· "양식 다운로드" 링크 클릭 → 위임장 양식(PDF) 다운로드
· 다운로드한 양식에 작성 후 스캔/촬영하여 업로드
· 허용 파일 형식: PDF, JPG, PNG (최대 10MB)
· 법인 대표자가 담당자에게 계좌 개설을 위임하는 서류
4
법인 주소 확인 서류
· 미첨부가 디폴트
· 허용 파일 형식: PDF, JPG, PNG (최대 10MB)
· 업로드 완료 시 파일명 + 삭제(X) 버튼 노출
· 법인 소재지를 확인할 수 있는 공공요금 고지서 또는 공적 서류
· 최근 3개월 이내 발행된 서류만 유효
5
본인 거주지 주소 확인 서류
· 미첨부가 디폴트
· 허용 파일 형식: PDF, JPG, PNG (최대 10MB)
· 업로드 완료 시 파일명 + 삭제(X) 버튼 노출
· 담당자 본인의 거주지를 확인하는 공공요금(전기, 가스, 수도) 고지서
· 최근 1개월 이내 발행, 성명과 주소가 명확히 기재되어야 함
6
제출하기 버튼
· 비활성이 디폴트 (회색 테두리)
· ①②③④⑤ 모두 파일이 첨부되면 활성화 (파란색)
· 클릭 시 서류 업로드 API 호출 → 성공 시 서류제출 완료 화면(SCR-DOC-099)으로 이동
· 업로드 실패 시: "파일 업로드에 실패했습니다. 다시 시도해 주세요"
선택에 따른 분기
· 다음 화면: 서류제출 완료 (SCR-DOC-099)
비고
· 이 화면은 해외 법인 사용자 전용 서류 제출 화면입니다.
· 일본 법인(SCR-DOC-003) 대비 ④법인 주소 확인 서류, ⑤본인 거주지 주소 확인 서류가 추가됩니다.
